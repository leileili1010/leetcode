# 277. Find the Celebrity

## Approach 1 - O(n) elimination approach (+ a final verification).

- A celebrity C satisfies:
    - Everyone knows C
    - C knows nobody
- We can eliminate non-celebrity candidates in one pass:
    - If knows(a, b) → a cannot be celebrity (celebrities don’t know others) → candidate = b
    - Else → b cannot be celebrity (everyone must know celeb) → candidate = a 
- After one pass you get one candidate.
    - But candidate may still be invalid. So we must verify them with a second pass.

```java
public class Solution extends Relation {
    public int findCelebrity(int n) {
        // Step 1: Find the only possible candidate
        int candidate = 0;
        for (int i = 1; i < n; i++) {
            if (knows(candidate, i)) {
                // candidate knows i -> candidate cannot be celeb
                candidate = i;
            }
            // else i cannot be celeb, keep candidate
        }

        // Step 2: Verify candidate
        for (int i = 0; i < n; i++) {
            if (i == candidate) continue;

            // Candidate must not know i
            if (knows(candidate, i)) return -1;

            // Everyone must know candidate
            if (!knows(i, candidate)) return -1;
        }

        return candidate;   // valid celebrity
    }
}
```
- time: O(n)
- space: O(1)


## Appraoch 2 - brute force (不推荐)

- Mark people who know someone as eliminated
- Track how many people know each person using a list
- Return the person who is not eliminated and is known by n–1 others

```java
public class Solution extends Relation {
    public int findCelebrity(int n) {
        boolean[] eleminated = new boolean[n];
        List<Integer>[] map = new ArrayList[n];

        for (int i = 0; i < n; i++) {
            map[i] = new ArrayList<>();
        }

        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if (i != j && knows(i, j)) {
                    eleminated[i] = true;
                    map[j].add(i);
                }
               
            }
        }

        for (int i = 0; i < n; i++) {
            if (!eleminated[i] && map[i].size() == n-1) {
                return i;
            }
        }
        return -1;
    }
}
```

- time：O(N^2)
- space: O(N^2)
