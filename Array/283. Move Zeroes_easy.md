# 283. Move Zeroes

[Leetcode 283 (Easy)][283]

[283]: https://leetcode.com/problems/move-zeroes/description/

## Approach 1
```java
class Solution {
    public void moveZeroes(int[] nums) {
        int i = 0;
        int n = nums.length; 
        for (int num:nums){
            if(num != 0){
                nums[i] = num;
                i++;
            }
        }
        while(i<n){
            nums[i] = 0;
            i++;
        }
    
    }
}
```
