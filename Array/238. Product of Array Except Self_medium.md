# 238. Product of Array Except Self

```java
// optimized approach
// use res for both prefix and postfix thus no need to have 3 arrays;
// time: O(n);
// time: O(1);
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int N = nums.length;
        int[] res = new int[N];

        int product = 1;
        for (int i = 0; i < N; i++) {
            res[i] = product;
            product *= nums[i];
        }

        product = 1;
        for (int i = N-1; i >= 0; i--) {
            res[i] *= product;
            product *= nums[i];
        }

        return res;
    }
}
```
