# 347. Top K Frequent Elements

### Appraoch 1 - bucket sort (peferred)

Follow up: Your algorithm's time complexity must be better than O(n log n), where n is the array's size.
- 统计每个元素出现频率 → freqMap 时间 O(N)
- 建一个桶数组 bucket[f]，里面存所有出现 f 次 的元素，索引代表出现次数
- 最多元素出现 N 次，因此桶数量 = N+1 时间 O(N)
- 从桶的末尾往前收集元素，收集 K 个即可 时间 ≤ N，因为所有元素最多走一遍
- time: O(N)

```java
class Solution {
    public int[] topKFrequent(int[] nums, int k) {
        // count freq of each letter
        Map<Integer, Integer> map = new HashMap<>();
        for (int num: nums) {
            map.put(num, map.getOrDefault(num, 0)+1);
        }
        int max = Collections.max(map.values());

        // build bucket
        List<Integer>[] buckets = new List[max+1];
        for (int key: map.keySet()) {
            int idx = map.get(key);
            if (buckets[idx] == null) buckets[idx] = new ArrayList<>();
            buckets[idx].add(key);
        }

        // update res   
        int[] res = new int[k];
        int idx = 0;
        for (int i = max; i >= 0 && idx < k; i--) {
            List<Integer> bucket = buckets[i];
            if (bucket == null) continue; // 千万不能忘记写
            
            for (int num: bucket) {
                res[idx++] = num;
                if (idx == k) return res;
            }
        }
        return res;
    }
}
```
