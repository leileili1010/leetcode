# 347. Top K Frequent Elements

### Appraoch 1 - bucket sort (peferred)

Follow up: Your algorithm's time complexity must be better than O(n log n), where n is the array's size.
- 统计每个元素出现频率 → freqMap 时间 O(N)
- 建一个桶数组 bucket[f]，里面存所有出现 f 次 的元素
- 最多元素出现 N 次，因此桶数量 = N+1 时间 O(N)
- 从桶的末尾往前收集元素，收集 K 个即可 时间 ≤ N，因为所有元素最多走一遍
- time: O(N)

```java
class Solution {
    public int[] topKFrequent(int[] nums, int k) {
        // contruct frequency map
        // bucket sort
            // An arrya of length of N+1
            // loop through map to get frequncy, 
            // arr[freq] = new ArrayList<>(), and then add nums
        // loop through arr from backwards, select k numbers
        Map<Integer, Integer> map = new HashMap<>();
        for (int num: nums) map.put(num, map.getOrDefault(num, 0)+1);

        int n = nums.length;
        List<Integer>[] freq = new List[n+1];
        for (int key: map.keySet()) {
            int idx = map.get(key);
            if (freq[idx] == null) freq[idx] = new ArrayList<>();
            freq[idx].add(key);
        }

        int[] res = new int[k];
        int idx = 0;
        for (int i = n; i >= 0 && idx < k; i--) {
            if (freq[i] == null) continue;
            for (int num: freq[i]) {
                res[idx++] = num;
                if (idx == k) return res;
            }
        }

        return res;
    }
}
```
