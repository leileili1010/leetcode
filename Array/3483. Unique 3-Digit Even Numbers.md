# 3483. Unique 3-Digit Even Numbers

## Approach 1 

### 写法一 - Array Iteration

枚举从 digits 中选三个数，分别作为个位数（必须是偶数）、十位数和百位数（不能是 0）。把生成的三位数加到一个哈希集合中。最后答案就是哈希集合的大小。

```java
class Solution {
    public int totalNumbers(int[] digits) {
        Set<Integer> set = new HashSet<>();
        int n = digits.length;
        for (int i = 0; i < n; i++) { // 个位数: 必须是偶数
            int a = digits[i];
            if (a % 2 > 0) {
                continue;
            }
            for (int j = 0; j < n; j++) { // 十位数：不能和个位数index相同
                if (j == i) {
                    continue;
                }
                for (int k = 0; k < n; k++) { // 百位数： 1）不能为0 2）不能和十位个位index相同
                    int c = digits[k];
                    if (c == 0 || k == i || k == j) {
                        continue;
                    }
                    set.add(c * 100 + digits[j] * 10 + a);
                }
            }
        }
        return set.size();
    }
}
```
- time: O(N^3)
- space: O(N)

### 写法二 - recursion

```java
class Solution {
    private boolean[] used;
    private Set<Integer> res = new HashSet<>();

    public int totalNumbers(int[] digits) {
        used = new boolean[digits.length];
        dfs(digits, 0, 0);
        return res.size();
    }

    private void dfs(int[] digits, int pos, int num) {
        // 递归出口：已经选了 3 位
        if (pos == 3) {
            res.add(num);
            return;
        }

        for (int i = 0; i < digits.length; i++) {
            if (used[i]) continue;

            int d = digits[i];

            // 百位不能是 0
            if (pos == 0 && d == 0) continue;

            // 个位必须是偶数
            if (pos == 2 && (d & 1) == 1) continue;

            used[i] = true;
            dfs(digits, pos + 1, num * 10 + d);
            used[i] = false; // 回溯
        }
    }
}
```
