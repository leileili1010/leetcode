# 859. Buddy Strings

这道题和1790的解答方法很类似

- time O(N)
- space O(1)

```java
class Solution {
    public boolean buddyStrings(String s, String goal) {
        // edge case 
        if (s.length() != goal.length()) return false;

        int n = goal.length();
        List<Integer> diff = new ArrayList<>();

        // case 1: s equals to goal
        if (s.equals(goal)) {
            int[] freq = new int[26];

            for (char c: s.toCharArray()) {
                freq[c-'a']++;
                if (freq[c-'a'] >= 2) return true; 
            }
            return false;
        }

        for (int i = 0; i < n; i++) {
            if (s.charAt(i) != goal.charAt(i)) diff.add(i);

            // case 2: have more than 2 different positions
            if (diff.size() > 2) return false;
        }

        // case 3: diff != 2, simply return false;
        if (diff.size() != 2) return false;

        // case 4: diff == 2
        int i = diff.get(0), j = diff.get(1);
        return s.charAt(i) == goal.charAt(j) && s.charAt(j) == goal.charAt(i);
    }
}
```
