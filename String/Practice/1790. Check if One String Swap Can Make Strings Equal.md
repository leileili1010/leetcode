# 1790. Check if One String Swap Can Make Strings Equal

1. 长度检查：确保两个字符串长度相等
2. 记录差异：遍历两个字符串，记录下有差异的索引位置
3. 判断条件：
    - 0个差异： 字符串相等，直接返回true
    - 2个差异：唯一允许的差异值，
        - 检查 s1 在这两个差异位置上的字符，与 s2 在这两个差异位置上的字符，是否能够通过一次交换互相匹配。
        - 即：s1[i] == s2[j] 且 s1[j] == s2[i]（其中 i 和 j 是差异索引）
    - 其他差异数 (1 或 > 2): 无法通过一次交换或零次交换实现相等，返回 False

```java
class Solution {
    public boolean areAlmostEqual(String s1, String s2) {
        // check if s1 and s2 have the same length;
        if (s1.length() != s2.length()) return false;
        
        List<Integer> diff = new ArrayList<>();
        int n = s1.length();

        // 找到不同的字母 记录index放入diff list
        for (int i = 0; i < n; i++) {
            if (s1.charAt(i) != s2.charAt(i)) diff.add(i);
            // case 1: diff > 2 不可能通过1个swap就能相等
            if (diff.size() > 2) return false;
        }

        // case 2: diff == 0 两个string相等
        if (diff.size() == 0) return true;

        // case 3: diff == 2 可能通过1个swap就能相等，但我们仍然要确认
        if (diff.size() == 2) {
            int i = diff.get(0);
            int j = diff.get(1);
            if (s1.charAt(i) == s2.charAt(j) && s2.charAt(i) == s1.charAt(j)) return true;
        }

        // case 4: diff == 1 不可能通过1个swap就能相等
        return false; 
    }
}
```

- time: O(N)
- space: O(1)
