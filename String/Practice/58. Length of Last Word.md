# 58. Length of Last Word

## Approach 1 - trim

- time : O(N)
- space: O(N), trim()会产生一个新string

```java
class Solution {
    public int lengthOfLastWord(String s) {
        s = s.trim(); // 去掉首尾空格
        return s.length() - s.lastIndexOf(' ') - 1;
    }
}
```

## Approach 2 - pointers (最优解)

- 时间复杂度：O(n)，其中 n 是 s 的长度。
- 空间复杂度：O(1)。

```java
class Solution {
    public int lengthOfLastWord(String s) {
        int i = s.length() - 1;
        while (s.charAt(i) == ' ') {
            i--;
        }

        int j = i - 1;
        while (j >= 0 && s.charAt(j) != ' ') {
            j--;
        }

        return i - j;
    }
}
```
