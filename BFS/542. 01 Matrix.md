# 542. 01 Matrix

## approach 1 - 从零出发找到其他点到零的路径

```java
class Solution {
    int[][] steps = {{1,0}, {-1,0}, {0,1}, {0,-1}};

    public int[][] updateMatrix(int[][] mat) {
        Deque<int[]> que = new ArrayDeque<>();
        int rows = mat.length;
        int cols = mat[0].length;

        // 把所有值为0的坐标放入队列，不是0的我们设置为最大值，需要更新
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                if (mat[i][j] == 0) que.offer(new int[] {i, j});
                else mat[i][j] = Integer.MAX_VALUE;
            }
        }

        while (!que.isEmpty()) {
            int[] cur = que.poll();
            int curRow = cur[0];
            int curCol = cur[1];
            
            for (int[] step: steps) {
                int row = step[0] + curRow;
                int col = step[1] + curCol;
                
                if (row < 0 || row >= rows || col < 0 || col >= cols || mat[row][col] <= mat[curRow][curCol] + 1) { // 如果下一步到0的距离已经比途经当下坐标到0的路径近，跳过不放入que
                    continue;
                }

                que.add(new int[] {row, col});
                mat[row][col] = mat[curRow][curCol] + 1; 
            }
        }
        return mat;
    }
}
```
- time: O(rows * cols)
- space: O(rows * cols)
