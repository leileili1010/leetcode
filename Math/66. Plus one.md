# 66. Plus One

## Approach
![alt text](image.png)

```java
class Solution {
    public int[] plusOne(int[] digits) {
        int N = digits.length;
        for (int i = N-1; i >= 0; i--) {
            if (digits[i] < 9) {
                digits[i]++;
                return digits;
            }
            digits[i] = 0;
        }

        // for loop 没有return 说明digits全是9
        int[] res = new int[N+1];
        res[0] = 1;
        return res;
    }
}
```

- 时间复杂度：O(m)，其中 m 是 digits 末尾 9 的个数。
- 空间复杂度：O(1)。返回值不计入。
