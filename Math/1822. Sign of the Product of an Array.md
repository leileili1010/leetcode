# 1822. Sign of the Product of an Array

```java
class Solution {
    public int arraySign(int[] nums) {
        int sign = 1;

        for (int num: nums) {
            if (num == 0) return 0;
            if (num < 0) sign = -sign;
        }
        return sign;
    }
}
```

这道题我们来研究一种错误的思路，先看这道题的限制条件：
- 1 <= nums.length <= 1000
- -100 <= nums[i] <= 100

如果数组有1000个数，每个数都是100，那么100^1000 就会overflow, 如果我们的思路是先计算total product 再判断符号就会出错
