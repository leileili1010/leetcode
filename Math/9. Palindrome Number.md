# 9. Palindrome Number

## Approach 1 

要求不能把数字convert to string 来做

暴力想法是，根据前置知识二，把 x 整体反转，得到 rev，然后判断 rev 和 x 是否相等。

但实际上，对于回文数来说，只需判断 x 的左半部分是否等于右半部分就行。
- 这道题注意数字有奇数个数位（例如12521）或者偶数个数位（1221）
- 我们用rev < x/10 来决定是否终止循环
    - 如果最后 rev == x, 说明是有偶数个数位
    - 如果 rev  == x/10, 说明是有奇数个数位


```java
class Solution {
    public boolean isPalindrome(int x) {
        // x < 0 or x为10的倍数，都不可能是palindrome
        if (x < 0 || x > 0 && x % 10 == 0) {
            return false;
        }

        int rev = 0;
        while (rev < x /10) {
            rev = rev * 10 + x % 10;
            x /= 10;
        }
        return rev == x || rev == x/10;
    }
}
```
