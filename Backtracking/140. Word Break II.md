# 140. Word Break II

## Approach 1 - backtracking

```java
class Solution {
    public List<String> wordBreak(String s, List<String> wordDict) {
        Set<String> wordsSet = new HashSet<>(wordDict);
        List<String> res = new ArrayList<>();
        dfs(s, wordsSet, res, 0, new StringBuilder());
        return res;
    }

    private void dfs(String s, Set<String> wordSet, List<String> res, int start, StringBuilder currSentence) {
        if (start == s.length()) {
            res.add(currSentence.toString().trim());
            return;
        }

        for (int end = start+1; end <= s.length(); end++) {
            String word = s.substring(start, end);
            if (wordSet.contains(word)) {
                int currLen = currSentence.length();

                currSentence.append(word).append(" ");
                dfs(s, wordSet, res, end, currSentence);
                currSentence.setLength(currLen);
            }
        }
    }
}
```
