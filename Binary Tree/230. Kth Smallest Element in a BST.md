# 230. Kth Smallest Element in a BST

这道题解题的关键在于 in-order gives sorted values in a BST.

## Approach 1 - DFS
- time: O(N)
- space: O(H)

### 写法一

```java 
class Solution {
    public int kthSmallest(TreeNode root, int k) {
        List<Integer> list = new ArrayList<>(); 
        dfs(root, list, k);
        return list.get(k-1);
    }

    private void dfs(TreeNode node, List<Integer> list, int k) {
        if (node == null) return;
        if (list.size() == k) return;

        dfs(node.left, list, k);
        list.add(node.val);
        dfs(node.right, list, k);
    }
}
```

### 写法二

```java
class Solution {
    private int ans;
    private int k;

    public int kthSmallest(TreeNode root, int k) {
        this.k = k;
        dfs(root);
        return ans;
    }

    private void dfs(TreeNode node) {
        if (node == null || k == 0) {
            return;
        }
        dfs(node.left); // 左
        if (--k == 0) {
            ans = node.val; // 根
        }
        dfs(node.right); // 右
    }
}
```


## Approach 2 -  iterative (推荐)

```java
class Solution {
    public int kthSmallest(TreeNode root, int k) {
        Deque<TreeNode> stack = new LinkedList<>();
        TreeNode cur = root;

        while (!stack.isEmpty() || cur != null) {
            while (cur != null) {
                stack.push(cur);
                cur = cur.left;
            }
            cur = stack.pop();
            if (--k == 0) {
                return cur.val;
            }
            cur = cur.right;
        }
        return cur.val;
    }
}
```

Recap - Iterative In-Order Traversal
1. 一直往左走 → 模拟 inorder(node.left)
2. 弹栈访问 → 模拟 visit(node)
3. 转向右子树 → 模拟 inorder(node.right)

```java
public List<Integer> inorderTraversal(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    Stack<TreeNode> stack = new Stack<>();
    TreeNode current = root;

    while (current != null || !stack.isEmpty()) {
        // Step 1: Go as left as possible
        while (current != null) {
            stack.push(current);
            current = current.left;
        }

        // Step 2: Process node
        current = stack.pop();
        result.add(current.val);

        // step 3: go right
        current = current.right;
    }

    return result;
}
```
