# 235. Lowest Common Ancestor of a Binary Search Tree

## Approach 1 - DFS

    TreeNode res;

    public TreeNode lowestCommonAncestor(TreeNode node, TreeNode p, TreeNode q) {
        bfs(node, p, q);
        return res;
    }
