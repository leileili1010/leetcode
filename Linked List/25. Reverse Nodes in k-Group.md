# 25. Reverse Nodes in k-Group

D (groupPrev) --> 1 (cur) --> 2 (temp) --> 3 (kth) --> 4 (groupNext & next) --> 5

1. 初始化
    - dummy node 为 groupPrev
    - 从groupPrev 出发找到kth节点, 当kth == null, break
    - kth节点后面就是groupNext
2. 不断反转 cur ----> kth这一段
3. 最后连上groupPrev & kth, 并且更新groupPrev (1)

```java
class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        ListNode dummy = new ListNode(-1, head);
        ListNode groupPrev = dummy;

        while (true) {
            // 每一次从groupPrev开始找kth node
            ListNode kth = findKth(groupPrev, k);
            if (kth == null) break;

            ListNode groupNext = kth.next;
            ListNode next = groupNext;
            ListNode cur = groupPrev.next;

            while (cur != groupNext) {
                ListNode temp = cur.next;
                cur.next = next;
                next = cur;
                cur = temp;
            }

            ListNode temp = groupPrev.next;
            groupPrev.next = kth;
            groupPrev = temp;
        }
        
        return dummy.next;
    }

    private ListNode findKth(ListNode head, int k) {
        ListNode cur = head;
        for (int i = 0; i < k && cur != null; i++) {
            cur = cur.next;
        }
        return cur;
    }
}
```

- time: O(N)
- space: O(1)
