# 451. Sort Characters By Frequency

## Approach 1 - maxHeap

```java
class Solution {
    public String frequencySort(String s) {
        // 1. 统计每个字母出现的频率
        int[] freq = new int[128];
        for (char c: s.toCharArray()) {
            freq[c]++;
        }
        
        // 2. 将每一个字母放入maxHeap, 按出现频率从大到小排序
        PriorityQueue<Character> maxHeap = new PriorityQueue<>((a,b) -> freq[b] - freq[a]); 

        for (int i = 0; i < freq.length; i++) {
            if (freq[i] > 0) {
                maxHeap.offer((char)i);
            }
        }

        // 3. 按频率的从大到小的顺序取出字母，组成string
        StringBuilder sb = new StringBuilder();
        while (!maxHeap.isEmpty()) {
            char c = maxHeap.poll();

            for (int i = 0; i < freq[c]; i++) {
                sb.append(c);
            }
        }
        return sb.toString();
    }
}
```

- time: O(n + k log k), k < 128 ==> O(n)
- space: O(k) → O(1) for maxheap, return sb is O(n)

## Approach 2 - bucket sort (最优解)

```java
class Solution {
    public String frequencySort(String s) {
        // 1. 统计每个字母出现的频率
        int[] count = new int[128];
        for (char c : s.toCharArray()) {
            count[c]++;
        }

        // 2. bucket sort
            // buckets 长度为s.length()+1, 因为字母出现的最大频率不会超过s.length()+1
            // index of buckets 代表频率
            // 把字母按频率放进对应的bucket
        List<Character>[] buckets = new List[s.length() + 1];
        for (int i = 0; i < 128; i++) {
            int freq = count[i];
            if (freq > 0) {
                if (buckets[freq] == null) {
                    buckets[freq] = new ArrayList<>();
                }
                buckets[freq].add((char) i);
            }
        }

        // 3. 从尾到头来遍历buckets, 依次取出bucket里面的字母
        StringBuilder sb = new StringBuilder();
        for (int f = buckets.length - 1; f > 0; f--) {
            if (buckets[f] != null) {
                for (char c : buckets[f]) {
                    for (int i = 0; i < f; i++) {
                        sb.append(c);
                    }
                }
            }
        }
        return sb.toString();
    }
}
```

- time: O(n )
- space: O(n)
