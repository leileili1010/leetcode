# 225. Implement Stack using Queues

## Approach 1 - two queues

1. Q2添加数字: 先把要添加的数字加入到Q2
2. 把所有Q1的数字放入Q2
3. 交换 Q1 and Q2

```java
class MyStack {
    Deque<Integer> que1;
    Deque<Integer> que2;

    public MyStack() {
        que1 = new ArrayDeque<>();
        que2 = new ArrayDeque<>();
    }
    
    public void push(int x) {
        que2.offer(x);
        
        while (!que1.isEmpty()) {
            que2.offer(que1.poll());
        }

        Deque<Integer> temp = que1;
        que1 = que2;
        que2 = temp;

    }
    
    public int pop() {
        return que1.poll();
    }
    
    public int top() {
        return que1.peek();
    }
    
    public boolean empty() {
        return que1.isEmpty();
    }
}
```

## Approach 2 - Follow up: only one queue (推荐，更加优化)

- time: O(N)ForPush,O(1)ForOthers
- space: O(N)

```java
class MyStack {
    Deque<Integer> que;
  
    public MyStack() {
        que = new ArrayDeque<>();
    }
    
    public void push(int x) {
        int size = que.size();
        que.offer(x);
        
        for (int i = 0; i < size; i++) {
            que.offer(que.poll());
        }
    }
    
    public int pop() {
        return que.poll();
    }
    
    public int top() {
        return que.peek();
    }
    
    public boolean empty() {
        return que.isEmpty();
    }
}
```
