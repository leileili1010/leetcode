# 1472. Design Browser History

## approach 1 by Lei - two Stacks

```java
class BrowserHistory {
    private Deque<String> backStack;
    private Deque<String> forwardStack; 

    public BrowserHistory(String homepage) {
        backStack = new ArrayDeque<>();
        forwardStack = new ArrayDeque<>();
        backStack.push(homepage);
    }
    
    public void visit(String url) {
        backStack.push(url);
        if (!forwardStack.isEmpty()) forwardStack.clear();
    }
    
    public String back(int steps) {
        while (backStack.size() > 1 && steps > 0) {
            forwardStack.push(backStack.pop());
            steps--;
        }
        return backStack.peek();
    }
    
    public String forward(int steps) {
        while (!forwardStack.isEmpty() && steps > 0) {
            backStack.push(forwardStack.pop());
            steps--;
        }
        return backStack.peek();
    }
}
```

- 另一種寫法

```java
class BrowserHistory {
    private String curr;
    private Deque<String> backStack;
    private Deque<String> forwardStack;

    public BrowserHistory(String homepage) {
        backStack = new ArrayDeque<>();
        forwardStack = new ArrayDeque<>();
        curr = homepage;
    }
    
    public void visit(String url) {
        backStack.push(curr);
        curr = url;
        forwardStack.clear();
    }
    
    public String back(int steps) {
        for (int i = 0; i < steps && !backStack.isEmpty(); i++) {
            forwardStack.push(curr);
            curr = backStack.pop();
        }
        return curr;
     
    }
    
    public String forward(int steps) {
        for (int i = 0; i < steps && !forwardStack.isEmpty(); i++) {
            backStack.push(curr);
            curr = forwardStack.pop();
        }
        return curr;
    }
}
```
