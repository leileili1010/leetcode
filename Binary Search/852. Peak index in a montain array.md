# 852. Peak index in a montain array


```java
class Solution {
    public int peakIndexInMountainArray(int[] arr) {
        int start = 0, end = arr.length-1;
        
        //每一次比较一定要chack index是否出界！！！
        while (start <= end) {
            int mid = start + (end - start)/2;
            if (mid+1 <= end && arr[mid] < arr[mid+1]) { // 说明在peak左边
                start = mid + 1;
            } else if ( mid-1 >= 0 && arr[mid] < arr[mid-1]) { // 说明在peak右边
                end = mid - 1;
            } else { // arr[mid] > arr[mid-1] && arr[mid] > arr[mid+1];
                return mid;
            }
        }
        return -1;
    }
}
```
