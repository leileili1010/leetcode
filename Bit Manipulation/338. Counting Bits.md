# 338. Counting Bits

```java
class Solution {
    public int[] countBits(int n) {
        int[] res = new int[n+1];

        for (int i = 0; i < res.length; i++) {
            res[i] = bits(i);
        }

        return res;
    }

    public int bits(int n) {
        int count = 0;

        while (n != 0) {
            if (n % 2 == 1) {
                count++;
            }

            n = n/2;
        }

        return count;
    }
}
```
